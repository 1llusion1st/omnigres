# Cluster views

This extension offers multiple views into the cluster to
provide most commonly requested information, for example:

* `job_labels`
* `jobs`
* `jobs_with_data`
* `namespaces`
* `namespaces_with_data`
* `node_cidrs`
* `node_images`
* `node_labels`
* `nodes`
* `nodes_with_data`
* `pod_ip_addresses`
* `pod_labels`
* `pods`
* `pods_with_data`
* `service_cluster_ips`
* `service_external_ips`
* `service_ip_families`
* `service_labels`
* `service_ports`
* `services`
* `services_with_data`
